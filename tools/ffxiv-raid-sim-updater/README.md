# ffxiv-raid-sim-updater

Or more commonly known as `updater` is a small helper program designed to extract downloaded simulator updates and replace the existing files with minimal user intervention.

---

## Features

- Extracts downloaded simulator updates and replaces the files automatically.

---

## Requirements

- 64-bit Windows or Linux.
- Installation of the main `raidsim` program
- Same as the main `raidsim` program.

---

## How to Use

Just build the tool and put it into the folder next to the raidsim executable and let the game use it.
