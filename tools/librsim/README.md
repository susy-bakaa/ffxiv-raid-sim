# librsim

`librsim` is a small utility library designed to acompany the Linux release of raidsim. It offers various extensions and features for the Unity runtime that are not natively supported by Unity itself.

---

## Features

- Set the Unity runtime window's title to custom text

---

## Requirements

Runtime

- 64-bit Linux
- Installation of the main `raidsim` program
- Same as the main `raidsim` program.

Building

- 64-bit Linux
- The following packages: 
-- gcc make cmake libX11-devel (Fedora)
-- build-essential cmake libx11-dev (Ubuntu)

---

## How to Use

Just build the library and put it into the editor in the plugins folder. To build this library use the included script.
